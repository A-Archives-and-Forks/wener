- 铅蓄电池 - 铅对环境有害、硫酸为腐蚀性液体
- 镍镉 有毒
- 镍氢 早起弱于 镍镉 - 但现在 镍镉 更好
  - 无毒 - 环保
  - 无记忆效应
  - 2005 年 低自放电镍氢电池/LSD NiMH
- 碱性电池
  - 一般为 锌-二氧化锰
- 锂离子电池
  - 无记忆效应
  - 需要额外保护电路
  - 成本高
  - 会自放电
- 锂离子聚合物电池
  - 成本低
  - 容量低
  - 可生产为不同形状
