# Media Glossary

- PTS - [Presentation timestamp](https://en.wikipedia.org/wiki/Presentation_timestamp)
- DTS - decoding time stamp
- I - Intra-coded picture - key frame
  - 不依赖前后帧进行压缩
- P - Predicted picture
  - 使用前面帧进行压缩
- B - Bidirectional predicted picture
  - 可使用前后帧进行压缩
- BDMV - blu-ray folder
