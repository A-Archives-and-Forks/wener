# DNS Service

- DNS 服务角色
  - AS - Authoritative Server
    - 提供域名 DNS 权威信息 - 域名信息源
  - recursive
    - 直接与 root dns 交互 - 递归查询
  - forwarder
    - 转发到最近 DNS - 提供缓存
