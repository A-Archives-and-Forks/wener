# Backstage

- [backstage](https://github.com/backstage/backstage)

  - platform for building developer portals
  - PostgreSQL, SQLite
- https://backstage.io/plugins

```bash
npx @backstage/create-app
cd my-backstage-app
yarn dev

yarn add --cwd packages/backend pg
```
