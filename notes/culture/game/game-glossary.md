- Bead Road
  - 这是百家乐中的一个记分工具，也称为 “珠盘路”。在珠盘路上，每个“bead”（珠子）代表一局游戏的结果。珠盘路会用不同颜色或符号来表示不同的结果，如“庄家赢”、“闲家赢”或“和局”。
  - 用途：玩家通过观察珠盘路上“beads”的排列和趋势，试图找出规律或模式，以帮助他们做出下一局的投注决定。

## Blackjack

| PlayType                | cn       | notes                                                                          |
| ----------------------- | -------- | ------------------------------------------------------------------------------ |
| Ante                    | 预注     | 玩家在开始游戏前的基本下注。                                                   |
| Ante Split              | 分牌预注 | 玩家在选择分牌时的额外下注。                                                   |
| Insure                  | 保险     | 如果庄家的明牌是A，玩家可以选择购买保险以防庄家有黑杰克。                      |
| Double                  | 加倍     | 玩家可以在初始两张牌上选择加倍下注，之后只能再拿一张牌。                       |
| Perfect Pair            | 完美对子 | 一种边注，玩家押注自己的两张牌会形成同花色的对子。                             |
| Twenty First Plus Three | 21+3     | 一种边注，结合了21点和三张牌扑克的玩法，玩家押注自己的两张牌和庄家明牌的组合。 |
| Side                    | 边注     | 这是指玩家在主游戏之外的额外投注，通常与主要游戏结果无关。                     |
| Side Split              | 分牌边注 | 当玩家选择分牌时，对每一手牌进行的边注。                                       |
| Side Insure             | 边注保险 | 对边注的保险投注，以防庄家有黑杰克。                                           |
| Side Double             | 边注加倍 | 在边注上加倍下注。                                                             |
