## Elasticsearch vs OpenSearch

- OpenSearch
  - Apache-2.0
  - 社区版 X-Pack
  - ⚠️ 开发很慢 https://github.com/opensearch-project/OpenSearch/graphs/contributors
    - 投入相对少得多
- Elasticsearch
  - ESLv2, SSPL
  - 收费 X-Pack
  - 更多插件支持
  - 基础 Security 需要用免费 License
  - 默认开启了 Phone Home
  - 最后 Apache 2.0 Elasticsearch 7.10
