# psql

| command   | for                      |
| --------- | ------------------------ |
| `\l`      | list database            |
| `\c <db>` | switch database          |
| `\dt`     | list public schema table |
| `\dt *.*` | list all table           |
| `\ds`     | list relation            |
| `\dn`     | list schema              |
