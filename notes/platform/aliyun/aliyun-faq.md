# 阿里云 FAQ

## ECS 按量 vs 包年包月

| 包年 | 折扣 | 按量 | 按量>包年 |
| ---- | ---- | ---- | --------- |
| 1 月 |      | 150% | < 60%     |
| 1 年 | 7.3  | 200% | < 47%     |
| 2 年 | 6.1  | 236% |
| 3 年 | 4.8  | 315% |
| 4 年 | 3.9  | 376% |
| 5 年 | 3.3  | 449% |

:::tip

- 按量 - 按时 - 后付费
  - 方便按需调整数量 - 动态扩缩容
  - 方便动态调整配置
  - 应付潮汐流量
  - 基于时间调整集群服务规模
- 买断 - 包年/包月 - 预付费
  - 价格优惠
  - 核心基础服务 - 例如： master、数据库

:::
