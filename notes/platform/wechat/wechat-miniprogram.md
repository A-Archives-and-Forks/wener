# 微信小程序

:::caution

- wxss
  - 不支持 `*`
  - 名字不能包含 `:`,`#`,`/`,`[`,`]`
  - 不支持有些场景的 `~`
  - 不支持 rem
- 运行时
  - 无 fetch
- 不支持 svg 标签
  - 支持 canvas
  - https://github.com/dntzhang/cax
    - 通过 canvas 渲染 svg
  - 可以通过 Image.src 使用 svg
    - https://juejin.cn/post/6936529740433997861
    - 可以 dataurl

:::

- https://mp.weixin.qq.com/cgi-bin/wx
