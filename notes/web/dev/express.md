# express

- [expressjs/express](https://github.com/expressjs/express)

# Version

## Express 5.0

- Node 18+
- `path-to-regexp@8`
  - `/*` -> `/*name`
  - `/{*splat}` 匹配 `/` 和 `/foo`
  - `/:file.:ext?` -> `/:file{.:ext}`
  - 不支持 regex
- https://expressjs.com/en/guide/migrating-5.html
