# CSS Rule

## 对其和排列

- {align,justify}-{items,self,content}
- align & justify
  - align - Block Axis
  - justify -  Inline Axis
- align-self 默认 stretch
- flex - flex-direction
  - justify - 主轴
  - align - 副轴
- alignment
  - 位置
    - center
    - start - 受 书写模式影响
    - end
    - self-start
    - self-end
    - flex-start
    - flex-end
    - left
    - right
  - 基线
    - baseline
    - first baseline
    - last baseline
  - 分布
    - stretch
    - space-between
    - space-around
    - space-evenly
- safe & unsafe
  - safe - 溢出时使用 start
