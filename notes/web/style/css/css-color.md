# CSS Color

- MDN [color-mix](https://developer.mozilla.org/en-US/docs/Web/CSS/color_value/color-mix)
- [css-color-mix](https://developer.chrome.com/docs/css-ui/css-color-mix)


```
color-mix(method, color1 [p1], color2 [p2])
```

- p1, p2: 0-100% - 默认 50%
  - 0% < p1+p2 <= 100%
