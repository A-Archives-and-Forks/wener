## HBuilderX

:::caution

- 构建 要求登陆+手机号认证 🤮
- 非常难用，建议只用于 调试
- 如想用其他 ide 开发 uni-app，只能使用cli模式

:::

- 写 uniapp 很难绕开 HBuilderX
- HBuilderX 创建的项目不同于 cli 创建的项目
  - 默认没有 package.json 文件
- cli
  - /Applications/HBuilderX.app/Contents/MacOS/cli
- 参考
  - dcloud
    - 开发者中心 https://dev.dcloud.net.cn/
