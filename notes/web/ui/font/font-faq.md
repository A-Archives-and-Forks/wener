# Font FAQ

## otf vs ttf

- ttf - TrueType Font
  - by 苹果
  - 20世纪80年代末
  - 兼容最好
- otf - OpenType Font
  - by 微软, Adobe
  - TrueType 的扩展
  - 支持 TrueType 和 PostScript 字体
  - 支持更复杂的排版功
    - 连字（ligatures）、样式集（stylistic sets）和多语言字符
