# HTML FAQ

## 响应式 {responsive-meta-tag}

```html
<meta name="viewport" content="width=device-width, initial-scale=1" />
```

- width - 设置宽度
  - 支持数字 - 单位 px
  - device-width=100vw,100%
- height
  - device-height
- initial-scale=1
- minimum-scale=0.1
- maximum-scale=10
- user-scalable=1
  - 0/1, yes/no
