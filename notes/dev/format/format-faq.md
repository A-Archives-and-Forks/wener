# Format FAQ

## Format vs Codec

- Format
  - 通常为一种容器
  - 容器变化不会导致实际内容变化
  - 容器格式
    - 脱离于数据
    - 提供元信息
    - 提供不同的访问方式
- Codec
  - 数据的一种呈现方式 - 此时和 Format 可互换
    - JSON、YAML
  - 信息 <-> 数据 转化的方式
    - 声音 -> 数据
    - 文字 -> 数据
    - 图像 -> 数据
  - 通常为二进制 - 信息结构严格
  - Codec 过程可能会有信息转换
