## Code Review

- 长期来看，提升代码质量
- 做好权衡
- 希望代码 stays consistent, maintainable
- 注意的点

  - 功能
  - 复杂度
  - 命名
  - 注释
  - 风格
  - 一致
  - 文档
  - 每行 - 确保能理解
  - 上下文
  - 好的点
  - 总结
