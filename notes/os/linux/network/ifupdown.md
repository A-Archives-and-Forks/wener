# ifupdown

- 是什么？
  - 网络配置调度器
  - 基于 /etc/network/interfaces 进行配置
  - 根据配置创建设备，调用配置脚本

- [ifupdown-ng/ifupdown-ng](./ifupdown-ng.md)
- ifupdown-debian
