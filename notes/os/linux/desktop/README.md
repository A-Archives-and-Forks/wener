# Desktop

- [Awesome](./desktop-awesome.md)
- [X11](./x11/README.md)
- WM - Window Manager
  - 管理窗口
    - 位置
    - 堆叠
    - 焦点
    - 输入
    - 热键
  - 渲染合并层
  - 会话
- DE - Desktop Environment
  - 提供桌面环境
  - 提供基础应用
    - archiver
    - media player
    - text editor
    - screenshot
    - photo vider

:::tip

- 没有 Window Manager 也能显示窗口内容，只是无法操作窗口

:::
