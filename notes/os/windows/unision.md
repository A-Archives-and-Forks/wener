# Unison

- 要求
  - Intel Evo 认证
    - 依赖 Intel Evo 厂商的优化 - 例如 WiFi, 功耗管理
    - 本身没有硬性 CPU 特性要求
  - 要求 Intel Twin Lake/13 代 CPU
  - 要求 Windows 11
- 主要特性 - iOS/Android
  - 文件传输
  - 将平板作为扩展屏
  - 相册
  - 电话
  - 短信
  - 通知
