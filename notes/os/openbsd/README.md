# OpenBSD

- [OpenBSD](https://en.wikipedia.org/wiki/OpenBSD)
  - 1995 forking NetBSD
- 关系
  - macOS 防火墙基于 OpenBSD PF
  - Android's Bionic C 基于 OpenBSD
  - LLVM 使用 OpenBSD 的正则库
  - Windows 10 使用 OpenSSH 和 LibreSSL
- 子项目
  - OpenSSH
  - LibreSSL - fork 自 OpenSSL 1.0.1g
  - OpenBGPD
  - OpenIKED
  - OpenOSPFD
  - OpenNTPD
  - OpenSMTPD
  - pf
