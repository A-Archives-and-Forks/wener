# pfctl

| cli       | for        |
| --------- | ---------- |
| pfctl -sr | show rules |
| pfctl -sa | show all |


- -s - show
  - queue
  - rules
  - Anchors
  - states
  - Sources
  - info
  - labels
  - timeouts
  - memory
  - Tables
  - osfp
  - Interfaces
  - all
