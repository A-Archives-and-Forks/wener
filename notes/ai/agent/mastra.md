# Mastra

- [mastra-ai/mastra](https://github.com/mastra-ai/mastra)
  - Apahce-2.0+EE, Typescript
  - 由原 Gatsby.js 团队创立的 TypeScript AI 代理框架
  - 核心模块：
    - **Agents**：拥有思考能力、记忆、工具调用的自治代理
    - **Workflows**：基于状态图结构 (Graph-based) 的持久化复杂编排流
    - **RAG**：内置高效的文档提取、向量化与检索管线
    - **Tools/Integrations**：支持通过 OpenAPI 迅速对接第三方 API，或编写自定义工具
    - **Evals/Observability**：内嵌 AI 特有的测试评估与可视化运行监控台
  - 底层基于 Vercel AI SDK 构建了智能模型路由
  - 定位：用熟悉的 TS 语法构建企业级 AI 应用，无需切换到 Python (LangChain/LlamaIndex) 生态
- 支持 workspace
  - skills
  - mcp
  - sandbox
