# V

- [vlang/v](https://github.com/vlang/v) 是什么？
  - v 之于 c 就是 ts 之于 js
  - MIT, V
  - V ~= Go + Rust
  - 跨平台 - 体验类似 Go - 支持移动端和 Web
  - 支持 wasm
  - 编译为 C - 支持 C 转 V
  - 热重载
  - 语言特性 - 默认不可变、无 null、无全局状态、并发
  - 内建 GUI 库
  - v run 可以交互式或以类似 shell 方式运行


:::caution

- v 看似很美好，但是很多承诺的功能并不完善
  - 例如最基本的内存管理 - 目前实现大多数场景会导致内存泄漏
  - v 有一点浮夸
- 开发者相对排外

:::
