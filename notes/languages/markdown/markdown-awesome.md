## 语法扩展

- Tables
- Fenced Code Blocks
- Footnotes - `[^1]`
- Heading IDs - `## Title {#custom-id}`
- Definition Lists - `:`
- Strikethrough - `~~X~~`
- Task Lists - `[ ]` `[x]`
- Emoji - `:joy:`
- Highlight - `<mark>`, `==marked==`
- Subscript - `H~2~O`
- Superscript - `X^2^`
- Automatic URL Linking
- Disabling Automatic URL Linking - `http://www.example.com`
- Admonitions - `:::`
  - docusaurus
