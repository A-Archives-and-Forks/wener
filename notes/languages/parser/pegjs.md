# PEG.js

- [pegjs/pegjs](https://github.com/pegjs/pegjs)
  - 最后更新 2019 - 活跃 fork [peggy](./peggy.md)
  - JavaScript PEG 语法解析生成器
- 语法
  - `''` - 单引号字符串忽略大小写
  - `""` - 双引号字符串大小写敏感
  - `!` `&` - 用于条件预测 - 类似于前向环视 - Lookahead
