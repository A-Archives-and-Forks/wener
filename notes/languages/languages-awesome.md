- [ballerina-platform/ballerina-lang](https://github.com/ballerina-platform/ballerina-lang)
  - Apache-2.0, Java
  - 面向云服务的编程语言
  - 内建 HTTP, HTTP2, gRPC, NATS 协议支持
  - 支持 xml, json 类型
  - 异步网络请求
  - 包含常用服务客户端 - twitter, graphql, openapi
  - 服务集成 - awslambda, azure_functions
  - 流程图形化 - 序列图, 函数图, 客户端对象图
