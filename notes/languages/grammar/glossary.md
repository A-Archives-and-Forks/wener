# Glossary

| Abbr. | Word                               | Desc                      |
| ----- | ---------------------------------- | ------------------------- |
| DFA   | Deterministic finite automaton     | 确定有限状态自动机        |
| CFG   | Context free grammar               | 上下文无关语法            |
| LL    | Left-to-right                      | 左到右                    |
| DCFG  | Deterministic Context Free Grammar |
| PEG   | Parsing expression grammar         | 解析表达文法 / since 2004 |

- Antlr 是 `LL(*)`
- `LL(*)` 是 DCFG 子集
- DCFG 是 CFG 子集
- PEG 类似 CFG 但并不是子集
