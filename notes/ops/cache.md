# Cache

| for                    | macOS                       | Linux                            | Windows             |
| ---------------------- | --------------------------- | -------------------------------- | ------------------- |
| `pnpm store path`      | $HOME/Library/pnpm/store/v3 | $HOME/.local/share/pnpm/store/v3 |
| `npm config get cache` | ~/.npm                      | ~/.npm                           | %AppData%/npm-cache |
