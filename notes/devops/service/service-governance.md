# Service governance

服务治理是指让企业内部服务之间达成协作能力。

- governance as a service - 治理即服务
- 作为 IT 和 业务部分 之间的调和
- 主要能力
  - 生命周期管理
  - 标准,策略
  - 最佳实践
  - 服务发现 - 服务仓库, 服务复用
  - 组织
  - 用户/团队 管理
  - API
  - 外部系统集成
  - 扩展
  - 依赖管理
  - 监控,审计
