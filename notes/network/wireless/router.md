# 无线设备

:::caution

- 尽量不要买过于 “智能” 路由 - 间谍
- 功能强大的路由不过是一个 嵌入式 的 Linux 系统
  - Linux 可很好完成 路由 功能 - 软路由
- 区分购买 路由 和 AP 功能/设备
- 无线 **没有** 有线快
- 尽量不要买 TP 的路由
- 天线数量 不影响信号
- 天线外观粗细 不影响信号

:::
