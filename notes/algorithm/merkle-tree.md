# Merkle tree

- [Merkle tree](https://en.wikipedia.org/wiki/Merkle_tree)
  - 墨克树/Hash tree/哈希树
  - branch, inner node = inode
  - by Ralph Merkle 1979
  - adopted by:  IPFS, Btrfs, ZFS, Dat, Git, Cassandra
  - 大多实现为二叉树
- Tiger tree hash
- 参考
