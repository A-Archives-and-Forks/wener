# Ceph Kubernetes
## Tips
* 建议每个节点相同性能配置
* 部署方案
  * Ceph 部署在集群外
  * Ceph 使用 [rook.io](https://rook.io/) 部署
  * 使用 [Helm](https://docs.ceph.com/docs/mimic/start/kube-helm/) 部署
* 参考
